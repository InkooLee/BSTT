Skewed tensor response model

For an order-KK tensor response Yi∈Rd1×⋯×dKYi∈Rd1×⋯×dK with a vector of covariates xi∈Rpxi∈Rp, we consider a tensor response regression model

Yi=B×¯(K+1)xi+Ei,  for  i=1,…,n, Yi=B×¯(K+1)xi+Ei,  for  i=1,…,n, 
where B∈Rd1×⋯×dK×pB∈Rd1×⋯×dK×p is an order-(K+1)(K+1) unknown tensor of regression coefficients, ×¯(K+1)×¯(K+1) is the (K+1)(K+1)-mode vector product, and the error Ei∈Rd1×⋯×dKEi∈Rd1×⋯×dK is an order-KK tensor. We model the skewness in the distribution of YY via
Ei=|Z2i|×KΛΛ+Z1i,Ei=|Z2i|×KΛΛ+Z1i,
 where ΛΛ=diag(λ1,…,λdK)∈RdK×dKΛΛ=diag(λ1,…,λdK)∈RdK×dK with skewness parameters λλ=(λ1,…,λdK)λλ=(λ1,…,λdK) and |M||M| denotes the matrix whose each element of is the absolute value of the corresponding element of matrix MM.

Tensor Spike-and-Slab Lasso prior

To address the desired dimension reduction of the regression coefficient tensor BB while ensuring practical clinical interpretation of covariate effects, we use βi1i2i3j=ηi3j+αi1i2i3jβi1i2i3j=ηi3j+αi1i2i3j to split the effect of jj-th covariate xjxj into overall effects η1jη1j and η2jη2j on two biomarkers and the difference αi1i2i3jαi1i2i3j of the effect of xjxj on biomarker i3i3 from site i2i2 of tooth i1i1. We use a common independent N(0,τ2)N(0,τ2) prior for each ηi3jηi3j and an independent spike-and-slab lasso based sparsity inducing prior for tensor A∈R28×6×2×6A∈R28×6×2×6 with elements αi1i2i3jαi1i2i3j as

αi1i2i3j∼iid(1−ωi1i2i3j)ψ(αi1i2i3j∣γ0)+ωi1i2i3jψ(αi1i2i3j∣γ),ωi1i2i3j∼iidBernoulli(ϕ),αi1i2i3j∼iid(1−ωi1i2i3j)ψ(αi1i2i3j∣γ0)+ωi1i2i3jψ(αi1i2i3j∣γ),ωi1i2i3j∼iidBernoulli(ϕ),
where γ0≪γ1γ0≪γ1 and ψ(u∣γ)=(γ/2)exp(−γ|u|)ψ(u∣γ)=(γ/2)exp(−γ|u|) is the mean-zero Laplace distribution. When γ≪γ0γ≪γ0, tensor spike-and-slab lasso prior closely retrieve to tensor spike-and-slab prior. The large value of ϕϕ produces highly sparse elements in AA.

Tensor Normal prior

The alternative choice for BB is tensor normal prior, B∼TN(0,bI28B∼TN(0,bI28, bI6,bI2,bI6)bI6,bI2,bI6) for some large b>0b>0.

GAAD data analysis using BSTT package

We fit the BSTT (Bayesian skewed tensor-tt) model with tensor spike-and-slab lasso (TSSL)/Tensor Normal (TN) prior.

## Load package
library(BSTT)

# n.burn = burn-in period, n.save = number of posterior samples, thin = thinning size
BSTT_TSSL <- BSTT_TSSL(Y,X,vecy, n.burn = 100, n.save = 1000, thin = 5)
BSTT_TN <- BSTT_TN(Y,X,vecy, n.burn = 100, n.save = 1000, thin = 5)
